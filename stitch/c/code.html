关于悬空指针与内存熵减的分析报告

悬空指针是指向已经释放的内存地址的指针。这在复杂系统中是导致崩溃和安全漏洞（系统熵增）的主要诱因。现代 C++ 通过 RAII 机制引入了智能指针，实现了内存所有权的自动化管理。

main.cpp C++ 20
#include <memory>

// 现代方式：使用 unique_ptr 确保单一所有权
auto node = std::make_unique<DataNode>(0x42);

// 当 node 超出作用域，内存会自动安全释放
// 彻底杜绝了手动 delete 遗漏导致的熵增
协议: unique_ptr

独占所有权，零开销抽象，不可拷贝只能移动。

协议: shared_ptr

引用计数机制，允许多个指针共享同一资源生命周期。
